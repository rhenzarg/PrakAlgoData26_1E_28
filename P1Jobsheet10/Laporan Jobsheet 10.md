## Pertanyaan
1. front dan rear bernilai -1 sebagai tanda bahwa queue masih kosong dan belum ada elemen sama sekali — indeks -1 tidak valid dalam array, sehingga digunakan sebagai penanda khusus.
   Sedangkan size bernilai 0 karena memang jumlah elemen yang tersimpan saat awal adalah nol.
2. Kode ini mengimplementasikan circular array. Jika rear sudah berada di posisi paling akhir array (max - 1), maka rear dikembalikan ke indeks 0 (awal array) agar slot kosong di depan bisa digunakan kembali.
3. Sama seperti nomor 2, ini juga bagian dari circular array. Setelah elemen terdepan dikeluarkan, jika front berada di posisi akhir array (max - 1), maka front dikembalikan ke indeks 0 untuk melanjutkan pembacaan dari awal array.
4. Karena queue bersifat circular, elemen pertama tidak selalu berada di indeks 0. Elemen terdepan selalu berada di posisi front, sehingga perulangan harus dimulai dari front agar semua elemen ditampilkan dengan urutan yang benar.
5. Kode ini digunakan untuk berpindah ke indeks berikutnya secara circular. Ketika i mencapai indeks terakhir (max - 1), operasi % max akan mengembalikan nilai ke 0, sehingga perulangan tidak keluar dari batas array.
